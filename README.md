## Part 2 – Coding Task

### Random Story Generator

Write a program that creates a funny random story.

### Step 1
Create three lists:

- A list of characters  
- A list of places  
- A list of actions  

Example:

characters = ["A monkey", "A robot", "A bird"]  
places = ["in the park", "at school", "on the moon"]  
actions = ["is dancing", "is sleeping", "is singing"]

---

### Step 2
Use `random.choice()` to pick:

- One character  
- One place  
- One action  

---

### Step 3
Print the full sentence.

Example output:

A monkey is dancing in the park!
